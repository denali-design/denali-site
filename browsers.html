Validation

Currently we validate our web components by visual inspection using Browser Stack which provides testing on 2000+ real devices and browsers, not emulators or simulators.

Since, our validation methods are done by human review there is more than likely a chance we missed something. If you are currently using a browser that has support and you find bugs please create issues here.
